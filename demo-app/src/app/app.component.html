fuzz-js

JavaScript fuzzy string search library. Finds in-exact string matches in any list of objects.

# Live Demo

Type something into the input field to see it in action.

Generate or use your own sample data in the "Input Data" tab.

Click on the "Search Options & Code Sample" tab to try some different options.

# Installing

npm install --save fuzz-js

For useage examples, take a look in the "Search Options & Code Sample" tab in the Live Demo above (link)
