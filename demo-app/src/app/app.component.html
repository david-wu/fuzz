fuzz-js

fuzz-js is an open source fuzzy string searching library written in JavaScript. It finds imperfect string matches inside a given array of objects.

# Advanced Usage

The "search" function returns additional search information. This includes the key of the matched string, edit distance, and a styled string based on the match ranges. It also accepts an options object for customizing the search. For a full list of options, go to the "Search Options" tab in the demo above.
